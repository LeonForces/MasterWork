3 слайд.  Целью работы является реализация методов детектирования и трекинга движущихся объектов для видеонаблюдения охраняемых зон и периметра аэропорта.
Из поставленной цели определяются следующие задачи:
Реализация метода детектирования с вычитанием фона для ночного видеонаблюдения охраняемых зон.
Реализация метода детектирования и распознавания беспилотников с помощью нейронной сети для охраны периметра аэропорта.
Подготовка датасета и обучение нейронной сети.
Реализация метода трекинга для нескольких объектов.

4 слайд. Метод детектирования с вычитанием фона основан на получении маски при вычитании двух последовательных кадров, избавлении данной маски от шумов, проявлении контуров движущихся объектов и обрамление рамками области детекции. Такой уровень детекции подходит для ночного охранного видеонаблюдения и не требует больших вычислених процессов.
